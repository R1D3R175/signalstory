# Installation

:::caution

signalstory is based on [signal](https://angular.io/guide/signals), and hence requires at least angular `16.0`

:::

Install using `yarn`, `npm` or `pnpm`:

```bash
yarn add signalstory
```

```bash
npm install signalstory
```

```bash
pnpm install signalstory
```
